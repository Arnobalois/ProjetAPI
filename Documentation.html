Documentation

Notre Api vous propose de rechercher un film dans la base de donnée , puis nous afficherons grace a l'api IMDB les notes attribuer par des site de critique de film et dans un second temps nous affichons le casting principale du film en question. Nous enregistrons en base de donnée le film que vous avez cherché et votre pays où vous vous situez.

En deuxième partie notre API vous propose de chercher le nombre de vue soit par pays soit par film

  • GET Nombre de vue avec le Nom du Pays

    exemple : on cherche les films recherché par des internautes en France sur l'API

    Endpoint : http://iparla.iutbayonne.univ-pau.fr/~abalois/WebService/ProjetAPI-main/Pays/index.php/?p=France

    Parametre : p (nom du pays)

  • GET Nombre de vue avec le Code du Pays

    exemple : on cherche les films recherché par des internautes en Belgique sur l'API donc code = BE

    Endpoint : http://iparla.iutbayonne.univ-pau.fr/~abalois/WebService/ProjetAPI-main/Pays/index.php/?c=BE

    Parametre : c (code du pays)

  • GET Nombre de vue avec le Titre du film

    exemple : on cherche la liste des pays des internautes qui ont recherché le film Titanic

    Endpoint : http://iparla.iutbayonne.univ-pau.fr/~abalois/WebService/ProjetAPI-main/Film/index.php/?t=Titanic

    Parametre : t (titre du film)
